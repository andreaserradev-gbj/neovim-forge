## UI Toggles on the Corne

All UI toggles use the `<leader>u` prefix (Space + u), then a letter:

| Toggle | Keys | Hand |
|---|---|---|
| Zen | Space + u + z | Left thumb + right |
| Format | Space + u + f | Left thumb + right |
| Spelling | Space + u + s | Left thumb + right |
| Wrap | Space + u + w | Left thumb + right |
| Line nums | Space + u + l | Left thumb + right |
| Relative nums | Space + u + Shift+l | Left thumb + right (shifted) |
| Diagnostics | Space + u + d | Left thumb + right |
| Hints | Space + u + h | Left thumb + right |

All are the same pattern: left thumb for Space, then two quick right-hand taps.

---

## Zen Mode for Presentations

Zen mode (`<leader>uz`) is perfect for screensharing or presenting. It hides:
- Side panels
- Line numbers
- Status bar
- Anything but the code

This is where keyboard-driven development shines: you're not juggling UI elements; you're just showing the code.

---

## Deployment Workflow

By Day 29, your entire workflow is:
1. Edit, test, commit (all keyboard-driven)
2. Push to GitHub
3. Vercel/Netlify auto-deploys
4. Site is live

Zero context-switching. Pure keyboard. Pure focus.
