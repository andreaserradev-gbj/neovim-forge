## Week 2 Recap: The Corne Advantage

By the end of Week 2, your Corne has become a seamless extension of your vim workflow. Let's recap how the Corne optimizes each skill:

---

## Week 2 Skills on the Corne

### Day 8: Leader Key
- **Left thumb presses Space:** Gateway to all leader commands
- **Right hand types the rest:** All secondary keys are comfortably reachable
- **Status:** Automatic muscle memory by now

### Day 9: Flash Navigation
- **Right hand (`s`):** Home row access, no left-hand involvement
- **Right hand (labels):** Continue with right hand
- **Ergonomic win:** Pure right-hand operation

### Day 10: Explorer
- **Left thumb + `e`:** Open explorer
- **Right hand (hjkl):** Navigate the tree using familiar motions
- **Status:** Seamless—your right hand is already trained for hjkl

### Day 11: Windows
- **iCtrl + hjkl:** Left inner key + right hand navigation
- **Status:** Perfect cross-hand coordination
- **Benefit:** Neither hand dominates, both stay positioned well

### Day 12: Picker
- **Left thumb:** Open picker with leader
- **Right hand:** Type the search query
- **iCtrl + n/p:** Navigate results with left inner + right hand
- **Status:** Balanced operation, both hands engaged

### Day 13: Surround
- **Right hand (`s`):** Activate surround
- **Right hand (rest):** Type the operation and target
- **Status:** Pure right-hand efficiency

## The Integrated Workflow on the Corne

A realistic Week 2 editing session on the Corne:

1. **Open editor:** `nvim .`
2. **Press Space f f:** Left thumb + right index (find files)
3. **Type search:** Right hand (use familiar typing position)
4. **Press Enter:** Right thumb (select file)
5. **Press `s`:** Right index (Flash activate)
6. **Type label:** Right hand (jump to target)
7. **Press Space - :** Left thumb + minus (create split)
8. **Press iCtrl j:** Left inner + right middle (focus lower window)
9. **Press Space /:** Left thumb + Raise + slash (grep)
10. **Type search:** Right hand (familiar position)
11. **Press Enter:** Select match
12. **Press s:** Right index (Flash, position text)
13. **Press s a i w ":** Right hand (surround the word)
14. **Press Escape (J+K):** Right index + middle (exit any mode)

This entire workflow uses the Corne's split design perfectly:
- **Left hand:** Leadership (Space) and modifiers (iCtrl, Shift)
- **Right hand:** Navigation (hjkl), targeting (Flash, labels), and editing (surround)
- **Thumbs:** Layer switching and Escape
- **No hand switching:** Neither hand leaves its half of the keyboard

---

## Muscle Memory Integration

By Day 14, all these motions should feel automatic:

- **Thumb → Space:** Precedes every leader command
- **Right hand → hjkl:** First instinct for navigation (explorer, windows, within files)
- **Right hand → `s`:** Flash is now second nature
- **iCtrl → hjkl:** Window navigation is practiced
- **Right hand → surround:** Pair manipulation is fluent
- **J+K → Escape:** Faster than physical Escape key

The Corne's design aligns with these automatic motions. You're not fighting against your keyboard—you're working with it.

---

## What Makes the Corne Special for Week 2

1. **Split hands = balanced load:** Neither hand dominates
2. **Home row optimization:** Most commands never leave home row
3. **iCtrl dedication:** Window navigation is timing-free
4. **Space for leader:** Left thumb is the perfect gateway
5. **Right-hand specialization:** Flash and surround are right-side operations

This is why Vim enthusiasts and split keyboard users overlap so much. The philosophy of both is: **minimize hand movement, maximize efficiency through composition.**
