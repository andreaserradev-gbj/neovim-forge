## Exercise 1: Copilot Workflow
Open a code file and:
1. Type a comment describing what you want to code
2. Let Copilot suggest the implementation
3. Review 3 suggestions
4. Accept one, refine with Vim edits

## Exercise 2: Testing Limits
Try Copilot on:
- Complex logic (see if it gets it right)
- Security-sensitive code (see how wrong it can be)
- Simple repetitive code (where it shines)

## Exercise 3: Documentation
Improve the AI features page with your own observations.

## Exercise 4: Reflection
When should you use Copilot? When should you code manually?
